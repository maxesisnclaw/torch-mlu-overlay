## Why this overlay exists

Cambricon's public `torch_mlu` source repository
(`https://github.com/Cambricon/torch_mlu`) last received meaningful
commits on **2025-03-15**, and the most recent publicly distributed
binary wheels for MLU370 are pinned to `torch_mlu 1.24.1 + torch 2.5.0`.
PyTorch upstream has since shipped `torch 2.10`.

This overlay forward-ports `torch_mlu` to newer PyTorch releases
(`torch 2.10` as of this writing) so that users running on the
publicly available MLU370 baseline can continue tracking upstream
PyTorch. We make no claims about Cambricon's internal roadmap or
product-line decisions.

2. **Build / apply / verification scripts** that we wrote.
3. **Smoke tests** that we wrote, calling public PyTorch APIs.
4. **Documentation** describing the porting work.

We do **not** redistribute:

- Cambricon-trademarked images or model weights.

To apply this overlay you must already legally possess a running
Cambricon `torch 2.5` base container, obtained directly from
Cambricon (their forum / official release channels).
